# GitHub Publishing Commands

# After creating your GitHub repository, run these commands:

# 1. Add your GitHub repository as remote (replace with your actual repo URL)
git remote add origin https://github.com/YOUR_USERNAME/pdf-table-extraction.git

# 2. Push your code to GitHub
git push -u origin main

# If you need to authenticate, GitHub will prompt you for:
# - Username: your GitHub username
# - Password: use a Personal Access Token (not your password)
#   Get token from: https://github.com/settings/tokens

# 3. After successful push, your code will be live at:
# https://github.com/YOUR_USERNAME/pdf-table-extraction
